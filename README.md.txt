# Text-to-Speech Desktop App

A local Python desktop application that converts text into natural AI speech.

## Features

- Neural text-to-speech voices
- 2 Options for Voice selection
- Desktop GUI interface
- Instant audio playback
- Built with Python and Tkinter

## Technologies

- Python
- edge-tts
- Tkinter
- sounddevice

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the app

python tts_app.py
